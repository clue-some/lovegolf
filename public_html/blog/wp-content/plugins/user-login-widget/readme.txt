== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `user-login-widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the widget in your templates trough the 'Appearance > Widget' menu in Wordpress
